\section{Experimental Setup}

\begin{sloppypar}

are fused each denoising step, then evaluated on \texttt{geneval},
\texttt{pickscore}, and \texttt{ocr} held-out sets.
Checkpoints: \texttt{jieliu/SD3.5M-FlowGRPO-Text} (OCR),
\texttt{jieliu/SD3.5M-FlowGRPO-PickScore},
\texttt{jieliu/SD3.5M-FlowGRPO-GenEval}.
Multi-teacher rows use \textsc{Avg} unless marked \textsc{PCGrad}.
Table~1: \textbf{bold} = single-teacher in-domain; gray = multi-teacher
in-domain; red/blue = change vs.\ max single-teacher baseline in that row.
OPD rows (\textit{peak}) use the maximum eval logged so far.
Route: $\checkmark$ = \texttt{teacher\_route\_by\_source}; $\times$ = off or not teacher-specific (e.g.\ \texttt{pw-avg}).
